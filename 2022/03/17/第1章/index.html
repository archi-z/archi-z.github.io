博弈论:第1章——收益

本文最后更新于：2022年3月21日 中午

第1章——收益

本章重点：

  • 收益矩阵
  • 囚徒困境
  • 严格占优策略
  • 四个重要的lessons

收益矩阵

根据博弈中两个玩家的策略，我们可以列出一个表格，表格的每行表示玩家1的一个策略，每列表示玩家2的一个策略。每个单元格中记录两个数字，代表玩家1，2在某种策略选择下的收益。第一个表示玩家1的收益，第二个表示玩家2的收益。

示例：

\(\alpha\) \(\beta\)
\(\alpha\) 0, 0 3, -1
\(\beta\) -1, 3 1, 1

玩家1和玩家2都有两个策略\(\alpha\)和\(\beta\)。当玩家1选\(\alpha\)，玩家2选\(\beta\)时，单元格中的记录为（3, -1），即在这种策略选择下，玩家1收益+3，玩家2收益-1。

Game1——绩点博弈

游戏规则介绍： 两名玩家进行博弈，每人均有两种策略，\(\alpha\)（内卷）或\(\beta\)（躺平），收益矩阵如下：

\(\alpha\) \(\beta\)
\(\alpha\) B-, B- A, C
\(\beta\) C, A B+, B+

上述收益矩阵仅描述了最终的结果，即绩点高低，并不是严格的收益（payoff）（收益不仅和绩点相关，它受多方因素影响），我们考虑两种不同类型的玩家，然后用数字来量化他们的收益。假设有两种玩家。（1）一种是自私的，他们仅关心自己绩点的高低，我们称之为“Evil git”；（2）另一种是道德的，他们会对自己造成他人绩点变低感到难过，也会对他人造成自己绩点变低感到生气，我们称之为“Indignant angel”。

这样，该博弈根据玩家的差异，可以将博弈划分为以下三种情况。

two “Evil git”——囚徒困境

Evil git仅关心自己绩点的高低，定义收益如下。

A为3，B+为1，B-为0，C为-1。

这种情况下，收益矩阵如下：

\(\alpha\) \(\beta\)
\(\alpha\) 0, 0 3, -1
\(\beta\) -1, 3 1, 1

从对手的策略来看，我的可能处境有两种，对手选\(\alpha\)，或对手选\(\beta\)。（1）如果对手选\(\alpha\)，我选\(\alpha\)收益0，选\(\beta\)收益-1，0>-1，这种情况下，我应该选\(\alpha\)；（2）如果对手选\(\beta\)，我选\(\alpha\)收益3，选\(\beta\)收益1，3>1，这种情况下，我还应该选\(\alpha\)。所以针对两个“Evil git”博弈的情况，双方都应该选择策略\(\alpha\)。我们将\(\alpha\)称为严格占优策略。

严格占优策略：不论对方采取什么策略，采用该策略的收益总大于采用其他任何策略。

lesson1:不要选择严格劣势策略。

问题：看起来（\(\beta\), \(\beta\)）的情况相较于（\(\alpha\), \(\alpha\)）的情况明显对双方都更有利，但为什么博弈的结果不会是（\(\beta\), \(\beta\)）？

这其中有两个方面的原因，（1）难以达成共识；（2）如果一方选择\(\beta\)，另一方没有理由不选择\(\alpha\)，因为这样收益更高。（其实根据纳什均衡，不难发现，这个博弈只有（\(\alpha\), \(\alpha\)）一个纳什均衡点，（\(\beta\), \(\beta\)）并不是纳什均衡点。）

lesson2:理性选择导致次优结果。

两个“Evil git”的情况就是典型的囚徒困境，跳出这个困境，无法靠参与博弈的玩家间彼此沟通实现，必须通过外部力量改变收益矩阵来跳出囚徒困境。

two “Indignant angel”

Indignant angel是道德的，定义收益如下。

两名玩家都选\(\alpha\)或都选\(\beta\)时收益同上。当一名玩家选\(\alpha\)另一名玩家选\(\beta\)时，选\(\alpha\)的玩家获得绩点A，本该收益3，但考虑到因为自己的选择让对方获得绩点C，内心的自我谴责令其收益-4，最终收益-1；而选\(\beta\)的玩家获得绩点C，本该收益-1，考虑到回家难以给家长解释，令其收益-2，最终收益-3。

这种情况下，收益矩阵如下：

\(\alpha\) \(\beta\)
\(\alpha\) 0, 0 -1, -3
\(\beta\) -3, -1 1, 1

按照上一种情况的分析思路，不难发现这种情况下并不存在严格占优策略。这种情况下，做出何种策略取决于不同的个人目的。如果希望保底结果不太差，这种玩家更可能会选\(\alpha\)；如果希望收益上限高，这种玩家更可能选择\(\beta\)。

lesson3:汝欲得之，必先知之。

两个“Indignant angel”的情况策略是不定的，而且需要玩家去和博弈的对手进行协调（coordinate），双方都选\(\alpha\)或者都选\(\beta\)，这种情况后续还会遇到，我们称之为协调博弈。

“Indignant angel” and “Evil git”

延续上述收益定义方式。

这种情况下，收益矩阵如下：

\(\alpha\) \(\beta\)
\(\alpha\) 0, 0 -1, -1
\(\beta\) -3, 3 1, 1

从“Indignant angel”的角度来看，并不存在严格占优策略。但从“Evil git”的角度来看，\(\alpha\)是他的严格占优策略，因此，“Indignant angel”也应该选择\(\alpha\)。

lesson4:换位思考。

一个“Indignant angel”和一个“Evil git”的情况下，“Indignant angel”单从自身角度考虑问题是不能确定策略的，但如果能够考虑到对手有一个严格占优策略\(\alpha\)，那么“Indignant angel”也就可以确定自己的策略应该选\(\alpha\)。

总结

收益矩阵在后续博弈中会频繁使用。

囚徒困境在现实场景有很多对应，军备竞赛、关税战、广告战。不能指望通过玩家间的沟通来跳出困境，达成合作，因为合作并非纳什均衡点，在利益驱动下，每名玩家都有背叛的动机。囚徒困境和协调博弈有别，后者可以通过玩家间的协调达成合作。

在有严格占优策略时，玩家肯定要选择严格占优策略。

lesson1:不要选择严格劣势策略。

lesson2:理性选择导致次优结果。

lesson3:汝欲得之，必先知之。

lesson4:换位思考。

其中lesson4很重要，因为换位思考其实是在分析自己可能的处境（也就是对手的策略），然后选择自己的应对策略。这在后续的博弈示例分析中还会频繁出现。
